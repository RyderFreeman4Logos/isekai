# 诸神乱舞的异世界中，被神厌弃的不可知论者想要爱智慧

**类型**：轻小说 / 异世界 / 科幻 / 智斗
**核心梗**：硬核程序员 vs 唯心魔法世界 / 热力学 vs 奇迹

## 写作风格指南
1.  **拒绝术语堆砌**：虽然主角是程序员，拥有底层视角，但不要满篇抛掷晦涩的理科术语。要用通俗、幽默、甚至带点哲理的比喻来解释（例如把魔法比作“人的愿望”，把魔力比作“世界分配的算力”）。
2.  **拒绝中二病**：主角是成年人，是理性的观察者。不要让主角通过喊招式名来强化气势，他的强大多体现在沉默的布局和对规则的利用上。
3.  **人物驱动 (Character-Driven)**：
    *   先构建鲜活的人物（性格、身世、愿望）。
    *   将人物放入剧情，推演他们会怎么做，而不是为了剧情强行降智。
    *   奥菲莉亚的伪装、尤娜的务实、沃卡的理智，都要通过行动体现。

## Meta 设定 (打破第四面墙)
1.  **留白**：正文中**不要**明说主角是“缸中之脑”或被军方带走。这应该是一个供读者挖掘的深层悬念。
2.  **舆论引导**：作者（我）会在读者论坛（或每章末尾的“作者有话说”里）开小号扮演“大聪明”读者。
    *   **作用**：用硬科幻理论（量子力学、模拟宇宙假说）去解释文中看似不合理的魔法现象，暗示世界本质。
    *   **效果**：增加互动感，提升作品逼格，引导核心粉丝挖掘真相。
