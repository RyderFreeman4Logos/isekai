# 佐藤直树 (Sato Naoki)

**真名**：佐藤直树
**异世界名**：和真名一样, 被称为Naoki
**种族**：人类（地球穿越者）
**前世**：普通大学生 / 体育爱好者
**阵营**：守序善（被误解为混乱邪恶）

## 核心设定
*   **外貌**：黑发短寸，体格健壮，面容憨厚，眼神清澈，一看就是个老实人。
*   **性格**：
    *   **敦厚老实**：典型的“滥好人”，信奉“吃亏是福”、“努力就会有回报”。
    *   **随遇而安**：穿越原因纯粹是因为转移协议写的这么有鼻子有眼觉得好玩，但他并不怨天尤人，反而既来之则安之。
    *   **纯情**：虽然为了恋人"开发"了某种意义上很“色情”的魔法，但本人也很纯情，是个只对莉莉姆一心一意的纯爱战神。
*   **被神厌弃的原因**：因为与被视为邪恶淫乱象征的魅魔相恋，被教会和主流社会视为堕落者。

## 魔法体系：风之触手 (Wind Tentacles)
*   **原理**：**体育系直觉**。他不进行复杂的魔力计算，而是将风元素视为自己手脚的延伸。
*   **表现**：由于长期的体育锻炼（篮球/足球等各种运动），他对“肢体”的空间感极强。他能像控制手指一样，同时操控数十根不可视的风之触手，精度达到微米级。
*   **应用**：
    *   **战斗**：全方位无死角的防御与控制。
    *   **生活/恋爱**：能够进行极高频率和精度的“按摩”，完美解决莉莉姆因魔化导致的过剩生理需求。

## 人际关系
*   **莉莉姆**：恋人。
*   **沃卡**：异界同乡/大哥。沃卡欣赏他的单纯，他视沃卡为可以交心的朋友。在沃卡离队时亲自送行。
