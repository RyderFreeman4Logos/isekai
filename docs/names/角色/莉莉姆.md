# 莉莉姆 (Lilim)

**种族**：魔族（魅魔）
**身份**：魅魔女王之女 / 魅魔公主

## 核心设定
*   **外貌**：虽然是魅魔，但气质更偏向于文艺少女，直到魔力失控时才会显露妖媚的一面。
*   **性格**：
    *   **纯爱向往**：并不歧视族人的生存方式，但自己因为看了太多人类世界的言情小说，希望自己能“一生一世一双人”。
*   **特殊体质**：**后天魔化**。
    *   并非基因导致魅魔族必须大量“补魔”，而是胎儿期受到外界魔法干涉(来自无意识释放魔法的魔族双亲)，加上在魅魔社会长大带来的身份认同，潜移默化地导致了慢性魔化(魅魔化)。
    *   普通男性的“效率”太低，无法填补她的空虚，直到遇到了拥有“风之触手”的直树。

## 人际关系
*   **佐藤直树**：唯一的恋人/解药。直树的存在让她可以在保持“纯洁”（只跟随一人）的同时活下去，是她的救赎。
