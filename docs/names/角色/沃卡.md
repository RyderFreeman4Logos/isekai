# 沃卡 (Voka)

**真名**：未知
**别名**：不可知论者 / 人生哲学家 / 非Admin
**种族**：人类（地球穿越者）
**职业**：软件工程师（前世） / D级冒险者 / 魔法研究者 / 幕后黑手 / 并**不**是管理员
**阵营**：守序中立

## 核心设定
*   **性格**：理性、冷静、讨厌麻烦。虽然看似冷漠，实则通过“社会工程学”手段在幕后修正世界。自认为是一名（人生）哲学家。
*   **动机**：对地球资源的资源匮乏和零和博弈感到绝望，试图在异世界尝试用魔法打破热力学第二定律（实现熵减），从而寻找人类文明的形态。
*   **信条**：不可知论。不否认神的存在，但拒绝盲目崇拜等基于情感和直觉的宗教思维，强调要证明神的存在需要无法辩驳的证据。

*   **能力体系 (Voka's Stack)**
    *   **魔力感知**: 对于这个世界的人来说，魔力感知为由于神经重组产生的**“第三只手”**（幻肢感）。没有数值化的蓝条，魔力耗尽表现为该肢体无力, 但不影响身体能力, 也不会有躯体症状。
    *   **魔法表现**: 信息操作 (Pointer Manipulation)。
        *   `Swap(A, B)`：无视距离交换物质坐标。由于不涉及物理搬运（不产生物理做功），**魔力消耗只与操作的信息量有关, 一般远低于正常空间魔法**。
        *   `Copy/Paste`：基于现成模板(某个实物)瞬间复制物质。消耗与质量成正比，和水系魔法师凭空制造水球一样，耗魔量与质量成正比，优点是能将复杂的东西作为一个集合进行复制, 但无法凭空制造没有模板的东西, 比如不能从一把6发子弹弹匣的左轮手枪复制出一把能装7发子弹的左轮手枪, 除非真有7发子弹弹匣的左轮手枪。
*   **箱庭世界 (The Box)**：
    *   **绝对防御**：将自身移至安全空间（海上基地），原地留下光学投影。
    *   **无限回廊**：将水放在首尾相接的空间中，形成无限回廊，困住敌人。
*   **化身系统**：
    *   **原理**：`Copy`制造完全一致的肉体 + 化身与本体身上的魔力膜进行神经信号的双向传递（Remote Reality）。
    *   **体验**: 奉行“信息即本源”，对化身无疏离感。
    *   **战术**：极度激进的“耗材流”打法。
    *   `Refresh(Part)`：利用源数据覆盖受损部位，实现瞬间“治愈”（其实是将化身刷新为本体最新状态）。
*   **重力井 (Gravity Well)**：
    *   `Loop(Bottom, Top)`：将空间底部的物质传送到顶部，利用重力无限加速。
    *   `Artificial Coriolis`：配合Loop与螺旋抽真空，制造无法逃逸的向心涡流。

## 人际关系
*   **尤娜**：被其救下的魔王之女，前期是可靠的同行者，后期逐渐相恋。
*   **奥菲莉亚**：互相利用。沃卡看穿了她的伪装，并成功说服她批准自己离队。
