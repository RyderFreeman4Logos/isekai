# 沃卡 (Voka)

**真名**：未知
**别名**：不可知论者 / 人生哲学家 / 非Admin
**种族**：人类（地球穿越者）
**职业**：软件工程师（前世） / 魔法研究者 / 幕后黑手
**阵营**：守序中立

## 核心设定
*   **性格**：理性、冷静、讨厌麻烦。虽然看似冷漠，实则通过“社会工程学”手段在幕后修正世界。自认为是一名（人生）哲学家。
*   **动机**：对地球资源的资源匮乏和零和博弈感到绝望，试图在异世界尝试用魔法打破热力学第二定律（实现熵减），从而寻找人类文明的形态。
*   **信条**：不可知论。不否认神的存在，但拒绝盲目崇拜等基于情感和直觉的宗教思维，强调要证明神的存在需要无法辩驳的证据。

## 能力体系 (Voka's Stack)
不拥有管理员权限（Admin）, 由于其真心相信世界可能是模拟出来的, 由于世界法则, 他的魔法表现为指针操作。
*   **指针操作 (Pointer Manipulation)**：
    *   `Swap(A, B)`：无视距离交换物质坐标。
    *   `Copy/Paste`：复制/粘贴物质（无法复制灵魂）, 消耗魔力比交换坐标大, 跟水系魔法师凭空制造水球一样, 耗魔量与质量成正比, 优点是能复制复杂的东西。
*   **箱庭世界 (The Box)**：
    *   **绝对防御**：将自身移至安全空间（海上基地），原地留下光学投影。
    *   **无限回廊**：将水放在首尾相接的空间中，形成无限回廊，困住敌人。
*   **化身系统**：通过以魔法为原理的全向跑步机和触觉反馈魔力膜远程操控复制体，本体永远躲在安全的海上基地。
