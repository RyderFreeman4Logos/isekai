# 奥菲莉亚·冯·罗德尔 (Ophelia von Rodel)

**身份**：罗德尔王国第二王女
**别名**：反派之一 / 戏精公主

## 核心设定
*   **外貌**：金发碧眼，精致如人偶，看似柔弱无害。
*   **性格**：
    *   **表象**：楚楚可怜、受惊的小鹿、心系国民/人类的公主。
    *   **本质**：腹黑、控制欲强、擅长操纵人心的权谋家。
    *   **目的**：利用召唤来的勇者巩固王权，或对抗其他势力。
*   **评价**：典型的“粉切黑”。

## 剧情表现
*   **初遇**：在沃卡苏醒时，伪装成走投无路、被迫召唤勇者的弱势公主，试图通过激起男性的保护欲来控制沃卡。
*   **互动**：
    *   在沃卡苏醒初遇时，伪装成弱势公主。
    *   **处理沃卡离队**：当沃卡以“能力不足”申请离队时，她毫不犹豫地批准并给予遣散费，展现出对“无用资产”的冷漠弃置（虽然表面上是仁慈）, 不过这也说不上有错，只是不符合她的人设。
