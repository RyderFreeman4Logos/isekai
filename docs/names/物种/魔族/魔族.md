# 魔族 (Demons)

**定义**：
1.  **地理/种族定义**：生活在魔族国家的人形智慧种族统称。
2.  **政治/宗教定义**（人类方）：除了人类之外，所有“非人”**或**“不信仰神”的智慧种族。包括兽人。

## 核心原理：魔化 (Demonization)
魔族与人类在基因层面的差异远小于表现型上的差异。所谓的“魔族特征”（如角、尾巴、特殊肤色等）并非完全由基因决定，更多源于**后天干涉**。

### 1. 胎儿期魔法干涉
*   魔族双亲（尤其是母体）在孕育胎儿时，会无意识地释放魔力，对胎儿进行“期望干涉”。
*   这种干涉类似于一种潜意识的“附魔”，将双亲对“魔族应该长什么样”的认知刻印在胎儿身上。
*   **高魔力抗性特例**：如果胎儿本身拥有极高的魔力天赋（高算力/高带宽），其自身的魔力场会无意识地抵御外界（包括父母）的干涉。因此，**越是高等的魔族，外表反而越接近人类**（返璞归真）。

### 2. 生长环境与认知
*   自我认同对魔族的生理结构有长期影响。在魔族社会长大，接受“我是魔族”的教育，会持续固化魔族特征。
*   **实验理论**：如果将一个魔族婴儿交由人类抚养，且不让其意识到自己是魔族，那么其魔族特征发育会显著弱化，甚至看上去和人类无异（此点为学界未公开的假说）。

## 社会结构
*   实力至上主义。
*   由于高等魔族外表接近人类，因此在魔族审美中，越像人类的外貌往往暗示着越强大的血统。
