# 魅魔 (Succubus)

**种族分类**：魔族亚种

## 种族特质
魅魔一族以“淫乱”和“可以通过性行为吸取能量”著称，但现代魔法研究表明，这更多是一种**种族特有的生理重构**，而非生存必需。

### 1. 魔化机制
*   **起源**：魅魔双亲在孕期会带有强烈的“魅魔就应该有很强的性需求”的潜意识预期，这种无意识的长期干涉通过魔力重塑了胎儿的内分泌系统。
*   **结果**：导致魅魔天生拥有极高的激素水平，表现为青春期后具有强烈的、难以抑制的生理需求。

### 2. 心理固化
*   魅魔社会的文化环境不断加深这种自我认同，导致这种生理特征被进一步固化。
*   这并非“不补魔就会死”，而是如果不进行高强度的释放，会查女生欲求不满的不适感。只要能忍耐或通过其他方式（如冥想/药物）疏导，理论上不需要依赖性行为生存。

## 个体差异
*   **低等魅魔**：完全被欲望和本能支配，外表特征明显（翅膀、尾巴、爱心眼）。
*   **高等魅魔**（如莉莉姆）：外表接近人类少女，保留了强大的精神系魔法天赋，能够一定程度上控制本能，但仍受生理缺陷的折磨。
