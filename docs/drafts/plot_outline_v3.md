# 异世界中长篇剧情大纲 V3 (Final Revision)

## 1. 核心设定 (Core Pillars)

### 1.1 主角定位：沃卡 (Voka) - "The Hacker"
*   **本质**: 并非系统管理员，而是唯一能利用物理引擎漏洞的**黑客**。
*   **性格**: 虚无主义哲学家，社恐/低情商。
    *   **优点**: 掌握绝对的物理真理，能解决一切“硬问题”（杀人、修路、治病）。
    *   **缺点**: 无法解决“软问题”（人心、政治、狂热信仰）。他需要盟友来翻译他的意图，否则在常人眼中他只是个冷漠的怪物。
*   **禁忌**: 严守“世界是虚拟的”这一秘密。因为一旦有人有了相同的信仰, 大概率不会爱这个世界和世界上的人。

### 1.2 核心矛盾
*   **启蒙 vs 蒙昧**: 沃卡试图传播“科学思维”（作为在这个世界生存的最优解），对抗教会的“神权解释”和联邦的“愚民政策”。
*   **不对称战争**: 沃卡拥有单体核武级的战力，但无法对百万被洗脑的平民下手。反派利用这一点进行道德绑架和政治围剿。

## 2. 角色阵营

### 2.1 主角团
*   **沃卡**: 物理输出 / 技术顾问 / 战略威慑。
*   **尤娜 (Yuna)**: 政治领袖 / 外交官 / 道德锚点。只有她能让沃卡这个“无情的解题机器”感受到人的温度。
*   **直树 (Naoki)**: 勇者 -> 叛逃者。因看透人类王国的腐朽，私奔至魔族，成为沃卡在此时期的重要接应人。
*   **魔族革新派**: 一群被主流社会排斥的亚人/魔族。

### 2.2 反派 A：利昂 (Leon) - 封闭的暴君
*   **定位**: 中期 Boss，物理上的阻碍。
*   **威胁**: **闭关锁国 + 钢铁洪流**。
    *   他切断了情报流通，导致内部发生的灾难（如大饥荒、瘟疫）外界不知情。
    *   利用“集体主义”将人异化为工具，制造了大量蒸汽时代的战争机器。
*   **结局**: 相对容易解决。一旦情报封锁被打破（如沃卡的卫星广播），加上沃卡的“斩首行动”，其统治会迅速瓦解。

### 2.3 反派 B：亚伯兰 (Abram) - 伪善的先知
*   **定位**: 最终 Boss，精神上的毒瘤。
*   **原型**: **现代邪教主 / 成功学大师**。
*   **话术风格**:
    *   绝不使用晦涩的经文，而是用“现代、通俗、看似科学”的语言（参考 `docs/ehit/讲道.txt`）。
    *   **伪逻辑**: “神是至高的小说家，世界是他的巨著。你不顺服神，就是破坏故事美感的错别字，注定要被修正。”
    *   **把爱当武器**: “神惩罚你是因为爱你，想让你悔改。”
    *   **去罪化**: “杀异教徒并不是杀人，而是帮他们重置操作系统。”
*   **威胁**:
    *   **解释权垄断**: 当沃卡展现神迹（如治好绝症），亚伯兰会说：“这是神借着异端的手在考验我们的信心。”或者“这是技术性调整。”
    *   **死忠肉盾**: 信徒是发自内心地维护他，甚至自愿为了“圣战”去死。沃卡杀不完这些人。

### 2.4 第三方势力：人类贵族 (The Aristocracy)
*   **定位**: 既是剥削者，也是统战对象。这方面由**尤娜**主导，沃卡负责物理站台。
*   **守旧派 (The Oppressors)**:
    *   **特征**: 视平民为牲畜，迷信血统论。往往与教会勾结。
    *   **处理**: **物理打服**。沃卡并不介意在这类人身上展示“天罚”（如：精准摧毁城堡的承重墙，让其在睡梦中塌房，物理意义上的）。
*   **开明派 (The Reformists)**:
    *   **特征**: 务实，对新事物（商业、技术）敏感，反感教会的过度敛财。
    *   **处理**: **利益捆绑**。通过黑潮港的贸易红利（如香皂、玻璃、通讯服务）将他们拉上战车，成为世俗王权对抗神权的盟友。

## 3. 剧情篇章 (Story Arcs)

### 第一卷：暗流与启程 (The Depths & The Journey)
*   **深海发育**: 沃卡在黑潮港周边建立深海基地，测试 Avatar。
*   **黑潮港相遇**:
    *   尤娜被教会追杀至此，沃卡出手相救（顺便做人体实验测试 Avatar 性能）。
    *   **启程**: 沃卡建立基础的情报“后门”后，因身份暴露风险及得知魔族那边有“有趣的同类”（直树），决定立刻动身护送尤娜前往魔族国家。

### 第二卷：魔域的变革 (The Network in the Wilderness)
*   **抵达**: 穿越封锁线，抵达魔族领地。发现这里并非地狱，但也处于相对落后且被人类国家严重封锁的状态。
*   **重逢**: 遇到直树。直树已是魅魔部落的“赘婿”，对此地了如指掌。
*   **合作模式**:
    *   **拒绝军事援助**: 沃卡明确拒绝直接提升魔族的军事科技（引发因果/战争责任）。
    *   **通讯革命**: 沃卡提供“全球通讯基站”（星链原型），以此换取在魔族领地的居住权。他的目的是打破信息壁垒，促进言论自由，让思想的火种自然传播。
    *   **最后保险**: 沃卡承诺留下“空间坐标”，在魔族面临灭族危机时，可以提供紧急的物理救援（带人跑路或定点清除威胁），但他不会指挥战争。
    *   **尤娜的定位**: 并非成为魔族的女王（那会成为固定的靶子），而是作为“游侠/外交官”，利用通讯网络整合各方资源，却不背负统治的沉重枷锁。

### 第三卷：铁幕被撕裂 (Breaking the Wall) - vs 利昂
*   **情报泄漏**: 沃卡的“卫星”监听到利昂国内的惨状（正在进行种族清洗或人体改造）。
*   **斩首行动**:
    *   沃卡不带军队，单枪匹马（Avatar * N）潜入利昂的钢铁都城。
    *   利用**光学迷彩**和**空间置换**，在万军丛中直接出现在利昂的卧室里。
    *   **对话**: 地球老乡见老乡。利昂试图用“为了大义”说服沃卡，被沃卡用虚无主义的冷笑驳回：“你只是在玩真人版《文明》玩上头了。”
*   **解放**: 利昂倒台，国门大开。但人们发现，心中的枷锁（宗教）比肉体的枷锁更难打破。

### 第四卷：神不在的星期天 (Godless Sunday) - vs 亚伯兰
*   **最终圣战**: 亚伯兰集结全人类国家的兵力，宣称要铲除“魔王（尤娜）”和“伪神（沃卡）”。
*   **舆论战**:
    *   亚伯兰全频道广播：“魔族的技术是偷窃了神的代码，他们会让世界蓝屏！”
    *   沃卡反击：不辩解，直接**公开展示“神迹”的原理**。现场直播解剖神职人员的“神力赐福”，展示那只是一种寄生虫或药物反应。
*   **决战**:
    *   战场不在前线，而在**神权中枢**。
    *   沃卡利用“物理规则漏洞”，当着全信徒的面，剥离了亚伯兰的“神格”（神器/系统权限），让他瞬间变成一个糟老头子。
    *   **诛心**: 并没有杀亚伯兰，而是让他以此生最丑陋的姿态活下去，看着由于科学普及，信徒一个个离他而去。

### 结局：文明的成年 (Adulthood)
*   世界没有毁灭，也没有重置。
*   人们开始学会：打雷是因为静电，生病是因为细菌，即使没有神，太阳照常升起。
*   沃卡和尤娜隐退（或许开了一家生意冷清的万事屋），偶尔出手修修这个世界的 Bug。
