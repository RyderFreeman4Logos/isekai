# 修改确认协议 (Modification Protocol)

为了确保剧情走向和世界观设定的准确性，避免与用户想法有明显差距，遵循以下修改流程：

## 适用范围
当计划修改或创建以下目录中的文件时：
*   `chapters/` (正文章节)
*   `docs/` (世界观设定、人物卡、时间线等)

## 执行流程
1.  **Implementation Plan**:
    *   在执行任何写操作（Write/Replace）之前，必须出 implementation plan。
    *   **语言要求**: 必须使用**中文**。
