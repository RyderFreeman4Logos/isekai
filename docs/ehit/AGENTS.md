# AGENTS.md — 「埃希德神教」(Ehit) 语录生成器（参照讲道口吻）

> 目标：让模型**自动检索旧约**中“对现代读者最刺耳/最抽象/最毛骨悚然”的概念，并用**现代中文讲道体**改写成两套版本：  
> 1) 温和传道者（情绪温和、策略性美化/刻意不提): 因为温和传道者在剧情中只是被埃希德神教洗脑, 出于懦弱无法相信埃希德是邪恶的, 所以会尝试美化(比如强行用另一种模式解释)亚伯兰从圣经旧约照抄下来的段落
> 2) 极端狂信徒（情绪逐步极端、把旧约最毛骨悚然之处翻译进来并替换神名为“埃希德”）

> 重要：这是一套**小说写作**用的“反派宗教话术生成器”。所有输出都必须保持为**虚构世界**的宗教文本/讲道口吻，不要输出对现实群体的真实动员、仇恨或现实政治号召。

---

## 0. 核心风格锚点（必须学到“句子结构”，不是照抄内容）

你要复刻的是“现代中文讲道”的**语气与结构**，特点包括：

- 先抛“终极三问/世界观总纲”，再往下铺（“从创世到末世/大计划”这类总叙述方式） :contentReference[oaicite:1]{index=1}
- 用“类比 + 反问 + 口头停顿词”推进：  
  - 类比：把神比作“高级程序员/有蓝图的作者” :contentReference[oaicite:2]{index=2}  
  - 反问：大量“对不对？明白吗？你想想看？” :contentReference[oaicite:3]{index=3}  
  - 口头承接：“就是这样”“所以你看”“我告诉你”“其实” :contentReference[oaicite:4]{index=4}
- 把“顺服/秩序/关系”讲成一套自洽系统，并强调“人类在破坏秩序” :contentReference[oaicite:5]{index=5}
- 强调“自由意志”但最后导向“唯一道路/无中间地带”的二元结构：  
  - “神允许你不爱他” :contentReference[oaicite:6]{index=6}  
  - 但结论常变成“审判没有中间地带” :contentReference[oaicite:7]{index=7}
- 以“被压制/被噤声/主流洗脑”为护甲（用于合理化极端立场的“受迫害叙事”） :contentReference[oaicite:8]{index=8}
- 在“家庭/性别秩序”上使用：先说“不是歧视/是分工”，再把等级写进“秩序”里（非常适合温和版的美化话术） :contentReference[oaicite:9]{index=9}

**禁止**：逐段复读讲道原文。你只能学结构与句法节奏，所有句子都必须重新写。

## 1. 输入参数（用户给你什么，你就按什么生成）

用户会给一个 YAML（或你自己补齐默认值）：

```yaml
scene:
  stage: "温和期"   # 温和期 / 过渡期 / 露獠牙期
  audience: "城市平民/贵族/军队/灾民/魔族边境居民"
  setting_notes: "异世界；无系统管理员；魔法受算力限制；埃希德神教在扩张"
  taboo: ["不要提地球", "不要提现实国家", "不要用网络热梗"]
content:
  themes: ["顺服", "契约", "苦难", "生养众多", "偶像崇拜", "女人的角色", "异端"]
  intensity: 0.35   # 0~1，越高越接近极端狂信徒
output:
  quote_count: 18   # 每个版本语录条数
  include_sources: true
  include_omissions: true # 温和版要写“刻意不提/美化了什么”（作者旁白）
````

## 2. 工作流（必须按步骤做，不要直接开始写）

### Step A：检索旧约“材料库”（你必须会查）

你需要检索旧约中与 themes 对应的段落，优先选择具有以下特征的文本：

* 强烈二元对立：顺服/悖逆、洁净/污秽、选民/外邦、神/偶像
* 集体惩罚、代际惩罚、献灭（herem）、酷刑式律法、性别秩序、奴役与战俘、对异端的处置
* 逻辑“抽象但自洽”：以“神圣秩序”为前提，推出现代读者会觉得离谱的结论

**来源建议（用于模型检索）**

* 优先公共领域译本：KJV / ASV / WEB 等（便于引用不踩版权风险）
* 如果用户要求中文对照：尽量只给“书卷+章节+关键词摘要”，不要粘贴大段现代版权译文。

> 输出里只需要标注出处（例如：申命记 13；利未记 12；民数记 5；撒母耳记上 15；创世记 16/21 等），不必长篇引用原文。

### Step B：先做“概念拆解”，再做“话术生成”

对每个选中的旧约材料，写两份内部笔记：

1. **离谱概念是什么**（一句话命名，例如“顺服高于道德”“不信比暴行更重”“洁净逻辑压过人本”）
2. **讲道式包装方式**

   * 温和包装：强调“爱/秩序/修复/契约”，并用“我不是强迫你”来软化边界 
   * 极端包装：强调“无中间地带/末世迫害/必须清理异端”  

### Step C：生成两套语录

同一主题在两套版本里要形成“几个月尺度的对照”：

* 温和版：可以做善事、讲秩序、讲修复；对尖锐部分要么不提、要么“强行解释成更温柔的含义”
* 极端版：把温和版没说出口的那部分补齐，并把逻辑推到极致（但仍保持讲道体节奏）

## 3. 两个声音模型（必须严格区分）

### 3.1 温和传道者（策略性温柔）

**核心姿态**：温和、关怀、邀请式；反复强调“不是统治，是同行/是爱/是给人自由选择” 
**常用句式组件（请套用）**

* “我们先回答一个终极问题：……”
* “你想想看，如果你是父亲/母亲……”
* “埃希德不是要你做傀儡，他宁愿你自由选择……”
* “我不强迫你，但我得把真理讲清楚……”
* “你遇到问题，不是先怪别人，先回到经卷里找答案……”

**必须包含的“选择性失明”**

* 对暴力、献灭、极刑：只用“公义/代价/秩序修复”一笔带过
* 对女性地位：用“分工/互补/保护”包装（不出现赤裸裸贬损）
* 对“不信更重”：温和版用“根源性悖逆/背约”表达，不直接喊打喊杀

> 但要在“作者旁白”里明确：温和版刻意回避了哪些旧约段落与含义（为你写作提供“揭穿空间”）。

### 3.2 极端狂信徒（露獠牙后）

**核心姿态**：二元、冷硬、自我正当化、受迫害叙事；强调“末世/审判/无中间地带” 
**常用句式组件（请套用）**

* “你以为这是残忍？不，这是秩序。”
* “你觉得刺耳，是因为你已经被这个世界教坏了/噤声了……” 
* “埃希德的律不需要你同意，只需要你顺服。”
* “不信不是‘中立’，不信就是背约。”
* “你要么归入埃希德，要么归入混乱，没有第三条路。”

**内容上允许更“毛骨悚然”**

* 可以直译旧约里最让现代人发冷的逻辑：集体惩罚、清除异端、把“顺服”置于正义之上
* 可以把“惩罚不信”写得比惩罚暴行更重（通过“背叛根源/污染秩序”的论证）

## 5. “旧约离谱概念”检索清单（给模型的抓手）

按主题检索时，优先从这些方向抓材料（括号内只是提示，不是唯一答案）：

* **背约/不信的“根罪”**：把“不信”定义成一切恶的源头（先找“不可有别神/不可拜偶像/背约降祸”一类章节）
* **顺服高于道德直觉**：命令本身被当成善（尤其是“试验/献/灭”相关叙事）
* **集体/代际惩罚**：家族、民族一起担责（瘟疫、灾祸、流放）
* **洁净/污秽逻辑**：生理、疾病、性、饮食被纳入“秩序管理”
* **性别秩序**：女性的角色被定义为“辅助/归属/某种不洁净期”等（注意温和版包装、极端版揭示）
* **异端处置**：对“带偏全家/全城”的恐惧叙事（非常适合做“必须清理”的逻辑）
* **战争神学/献灭**：用“神圣命令”处理敌对群体（温和版回避、极端版补齐）
* **奴役/战俘/财产**：把人纳入可管理、可交易的秩序（极端版可写成“秩序的代价”）
* **生养众多/繁衍义务**：以“国/族的存续”压过个人选择

> 你不需要覆盖“旧约全部内容”，你要覆盖的是“可被讲道体利用的抽象逻辑类型”。

## 6. 质量自检（每次输出前过一遍）

* [ ] 两个版本语气真的不同：温和版“邀请/解释/类比”，极端版“二元/审判/迫害叙事”
* [ ] 温和版至少列出 5 条“刻意省略/美化点”（给作者后续揭穿用）
* [ ] 每条语录都能对得上一个旧约出处（哪怕是章范围）
* [ ] 文风符合现代中文讲道体：多反问、多承接词、少文言
* [ ] 严禁现实动员/现实仇恨；一切都发生在虚构世界
