# 剧情时间线 (Timeline)

## 00000001_第1章 这个物质资源有限的世界实际上是个地狱
*   **地点**: 地球 -> 虚空/梦境
*   **关键事件**:
    *   沃卡对地球的内卷与资源匮乏感到绝望。
    *   收到来自 `Project_Isekai` 的神秘邮件，签署《异世界转移志愿协议》。
    *   睡着后失去意识（推测为被麻醉气体麻醉）。
*   **伏笔**: 协议中关于“转移要求”的免责条款。

## 00000002_第2章 欢迎来到异世界
*   **地点**: 罗德尔王国·王宫召唤室旁客房
*   **关键事件**:
    *   沃卡苏醒，发现自己身处奢华的异世界。
    *   不仅没死，反而被视为“勇者”召唤。
    *   遇到第二王女奥菲莉亚，猜测她可能是伪装（“粉切黑”）。
*   **新登场**: 奥菲莉亚·冯·罗德尔。
