# 罗德尔王国 (Rodel Kingdom)

**类型**：君主制国家
**当前统治者**：罗德尔王室（奥菲莉亚之父）

## 概况
*   **地理位置**：位于人类领土，可能是对抗魔族的前线。
*   **政治状况**：
    *   表面光鲜，内部可能面临魔族压力或贵族腐败。
    *   通过召唤异世界勇者来补充战力。
*   **与主角关系**：沃卡的降临点。

# 埃希德神教 (Ehit Religion)

**类型**：宗教组织 / 神权政权
**教宗**：亚伯兰 (Abram)

## 核心教义
*   **崇拜神祇**：埃希德 (Ehit)。
*   **排他性**：视无神论者、自然神论者、沃卡为异端/恶魔。
*   **社会结构**：严格的等级制度，神职人员拥有绝对生杀大权。
