# 聊天摘录：魔族 / “超越者”概念（原文大段摘录 + 助手论文草稿）

## 你提出的设定：魔族=超越者（带贬义/中性对照）
我想再谈谈魔族的设定。就是，我打算在剧情中引入一个新概念，那就是魔族是一个有点带贬义的说法，如果我们选择用好听一点的说法，实际上可以叫他们“超越者”。意思是，原本作为人类的人性的某一部分或者大部分崩坏之后，可以被认为是超越了原本的人性(部分挣脱了人性的枷锁/框架)，从而能做到之前做不到的事。也就是说，因此会带来正向的影响和副作用。

而“超越者”这个概念衍生，其实挺有意思的，可以类比于尼采的超人哲学。当然，不同的超越者，超越的东西也不一样。有些人超越的是原生家庭，或者原生的民族、文化、国家认同，还有些人超越的是某些传统观念，比如男女之间不平等的各种社会契约、社会规范，还有些人超越的是人类的某些欲望，比如，就像美国国父那样，由于他们超越了某些欲望，所以他们能做出伟大的选择。

我希望探讨的是超越者可以有哪些细化的类型，也就是说，他们分别超越了人性中的哪些要素，或者说，他们的人性中哪些部分崩溃了，以及这之后衍生出来的正向效果和副作用。要尽可能多地列举每一种超越者带来的结果

提示: 作品中基本不会有时间魔法和精神干涉魔法, 而且魔族虽然各不相同, 但面对人类的威胁是会团结起来对抗的(因为忠于人性的人类很难认同人性部分崩坏的魔族)

可以, 但也要现实一点, 毕竟疯得太厉害的群体往往不能稳定存续(如果这一模因带来的优势不足以抵消劣势, 会导致无法繁衍或没有生力军加入), 这种魔族就没必要写了, 因为这种魔族只会在历史长河中昙花一现, 或者连在魔族内部都无法容忍(比如过于残忍/不可控/容易背叛友军)

另外这个世界没有系统管理员这种角色, 我不打算写(主角也只是个能力有限的漏洞利用者, 不是管理员)
