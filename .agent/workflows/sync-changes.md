---
description: 同步用户的手动更改到当前会话
---

此工作流用于在用户进行了手动编辑（通过外部编辑器或终端）后，帮助 Agent 快速理解发生了什么变更，以便同步上下文。

## 1. 分析最近提交
运行以下命令查看最近的 5 次提交，了解变更概况：
```bash
git log --oneline -n 5 --stat
```
// turbo

## 2. 检查当前工作区状态
查看当前是否有未提交的修改：
```bash
git status -s
```
// turbo

## 3. (可选) 查看具体差异
如果需要深入了解某次特定提交或当前变更的细节，请根据上述输出运行 `git show <commit_hash>` 或 `git diff <file_path>`。
特别是涉及 `docs/` 或 `chapters/` 的变更，请务必仔细阅读以更新你内部的世界观模型。
