description: 新会话初始化 - 快速建立作品认知
---

此工作流用于在开启新会话时，快速读取核心设定文件和最新进度，以便立即进入写作状态。

## 1. 读取核心规则
首先，了解写作风格和世界观更新规则：
- 读取 `.agent/rules/writing_style.md` (写作风格指南)
- 读取 `docs/timeline.md` (当前剧情进度与伏笔)

## 2. 读取核心设定
建立对世界的基础认知（按需读取，若文件过大可只读目录或 summary）：
- 读取 `docs/01-world-md-原始设定集.md` (底层逻辑)
- 读取主角档案: `docs/names/角色/沃卡.md`
- 读取关键角色档案 (视当前剧情涉及人物而定，如奥菲莉亚、尤娜、新角色等)

## 3. 确认当前进度
找到最新的章节文件（通常在 `chapters/` 目录下编号最大的文件），并读取其全文，以确认上一章的断章位置和剧情状态。
```bash
ls -1 chapters/ | sort | tail -n 1
```
// turbo

## 4. 准备就绪
基于以上信息，你就拥有了继续创作所需的上下文。无需读取所有文件，避免上下文溢出。
