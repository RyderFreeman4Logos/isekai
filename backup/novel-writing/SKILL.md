# 异世界小说写作指南

## 核心原则

1. **Show > Tell**：通过场景展示，而非直接告诉读者
2. **多感官描写**：视觉、嗅觉、触觉、听觉、温度
3. **心理层次**：即使理性角色也需要情感波动的描写
4. **对话有个性**：加入语气、动作、神态，不要纯功能化

## 写作前检查清单

开始写作前，确认已读取：
1. `.agent/rules/writing_style.md`（写作风格）
2. `docs/timeline.md`（剧情进度）
3. 最新章节（确认断章位置）

## 写作流程

### 1. 场景开头

```
✓ 好的开头：
"凌晨四点。沃卡坐在有些发旧的人体工学椅上，听着楼下街道传来的轰鸣声。"
→ 时间+角色+感官细节

✗ 避免：
"沃卡是一个理性的人，他不喜欢这个世界。"
→ 直接告诉读者角色特点
```

### 2. 环境描写

每个新场景必须包含至少2种感官：

| 感官 | 示例 |
|------|------|
| 视觉 | 光线、色彩、物体形态 |
| 嗅觉 | 血腥味、花香、汗臭 |
| 触觉 | 丝绸的顺滑、地面的冰冷 |
| 听觉 | 脚步声、风声、心跳 |
| 温度 | 阴冷、灼热、潮湿 |

### 3. 心理描写

即使主角非常理智，也需要：
- 情感的存在（哪怕是生理意义上的, 并且被系统2习惯性抑制）
- 生理反应（心跳、呼吸、肌肉紧绷）
- 内心的权衡过程

```
✓ 好的写法：
"沃卡的冷漠地看着痛得在地上打滚的勇者，虽然很可惜, 但既然对方执意要为异端审判所服务，那么他不会给予对方丝毫同情。"

✗ 避免：
"沃卡毫无情绪地看着这一切。"
```

### 4. 对话处理

每段对话应包含：
- 说话内容
- 说话方式（语气、声调）
- 伴随动作或神态（可选但推荐）

```
✓ 好的对话：
"真的不留下来吗？"直树背着一把比他还高的大剑，神色有些落寞。

✗ 避免：
"真的不留下来吗？"直树问。
"不了。"沃卡回答。
```

### 5. 转场处理

- 场景切换使用 `***` 分隔
- 时间跳跃需要铺垫，不要突然"三个月后"
- 大时间跳跃可用简短的过渡段落

## 章节格式

- 文件名：`{n:08}_第{n}章 {章节名}.md`
- 不使用 H1/H2 标题
- 段落间空一行
- 长度：2000-5000字
