# 写作反模式清单

本文档列出AI写作常见的问题模式，写作时必须避免。

## 1. Tell > Show（直接告诉而非展示）

### 问题
直接用作者视角评价角色/局势，而非通过场景让读者自己发现。

### 错误示例
```
这种为了激发雄性生物保护欲而精心设计的"弱势姿态"，虽然略显刻意，
但考虑到她的身份和年龄，这恰恰证明了她是一个极其厉害的政治家。
```
→ 直接告诉读者"她在演戏"

### 正确写法
```
奥菲莉亚眼角的泪珠要落不落。但当沃卡转身去看窗外时，
他的余光捕捉到——那滴泪在一瞬间被不知名的力量高速蒸发，
取而代之的是一道审视的目光。
```
→ 通过细节让读者自己发现

---

## 2. 心理描写扁平化

### 问题
主角过于理性，像机器人而非人类。缺少情感波动和内心挣扎。

### 错误示例
```
沃卡平静地接受了现状。既没有尖叫，也没有掐自己大腿确认是否在做梦。
```

### 正确写法
```
沃卡深吸了一口气。
理智告诉他应该平静接受——毕竟这正是他签署协议时预期的结果。
但当陌生的天花板真实地映入眼帘时，他还是感到一阵轻微的眩晕。
不是恐惧，更像是……一种终于落地的释然，混杂着对未知的本能警觉。
他花了三秒钟才把这些情绪压回心底。
```
→ 即使压制情感，也要展示压制的过程

---

## 3. 环境描写单一化

### 问题
只有视觉描写，缺少其他感官。环境描写功能化，只交代必要信息。

### 错误示例
```
城外，迷雾森林边缘。
沃卡站在一棵巨树的枝干上，俯视着下方那头正在啃食树皮的庞然大物。
```

### 正确写法
```
城外，迷雾森林边缘。

晨雾尚未散尽，空气中弥漫着腐叶和苔藓特有的潮湿气息。
沃卡蹲伏在一棵巨树的枝干上，树皮粗糙的纹理硌着他的膝盖。
耳边是林中此起彼伏的虫鸣，偶尔夹杂着某种大型生物撕扯树皮的钝响。

那是一头剑齿熊。正专注于啃食面前的老树，浑然不知有个比魔物还危险的东西正从上方窥视。
```
→ 嗅觉+触觉+听觉+视觉

---

## 4. 对话功能化

### 问题
每句对话都在推进设定或剧情，缺少人物个性和自然互动。

### 错误示例
```
"在这个世界，魔法并非凭空而来。"罗瑟琳说道。
"它是神明对信徒的恩赐，是信仰在现实世界的投影。"
```

### 正确写法
```
"在这个世界，魔法并非凭空而来。"
罗瑟琳推了推眼镜，目光扫过面前这群来自异界的年轻人。
她的语气像是在陈述一个无需质疑的公理。
"它是神明对信徒的恩赐。"
她顿了顿，似乎在斟酌措辞。
"……当然，在座各位来自无神的世界，对这一点可能需要时间接受。"
```
→ 加入动作、停顿、语气变化

---

## 5. 时间跳跃突兀

### 问题
大段时间被一句话带过，读者缺少过渡感。

### 错误示例
```
三个月过去了。
直到那个夜晚。
```

### 正确写法
```
那封邮件发出去后，就像石沉大海。

一天，两天，一周。

沃卡试图不去想它。代码依然要写，Bug依然要改，
那个虽然未必会出事故但绝对不允许停下来的世界，依然在按部就班地运转。
那个深夜的冲动，逐渐被忙碌的日常掩埋。

三个月。

直到那个夜晚——
```
→ 用节奏感和日常片段铺垫时间流逝

---

## 6. 战斗描写技术化

### 问题
战斗更像技术文档或测试报告，缺少紧张感和情感投入。

### 错误示例
```
"骨骼强度符合预期。'低通滤波器'工作正常。"
他冷静地记录着结果，仿佛断掉的不是自己的手。
```

### 正确写法
```
咔嚓。
清晰的骨折声在静谧的林中格外刺耳。
沃卡低头看了一眼——左小臂呈现出一种诡异的扭曲角度，
骨茬几乎要刺破皮肤。

但痛觉没有传来。魔力膜忠实地过滤掉了所有过强的神经信号。
只剩下一种钝钝的压迫感，像是被人用力捏住了手臂。

"有意思。"
他低声自语，声音平静得有些诡异。
那头剑齿熊似乎被这种不合常理的反应吓到了，
后退了半步，发出低沉的威胁性咆哮。
```
→ 加入感官描写、环境反应、节奏控制

---

## 7. 评价性语言过多

### 问题
使用"太完美了"、"极其厉害"、"简直"等评价性词汇。

### 错误示例
```
太完美了。
无论是那恰到好处的颤音，还是眼角那滴倔强着不肯落下的泪珠，
都像是经过了无数次排练一样精准。
```

### 正确写法
```
沃卡看着她，沉默了两秒。
那恰到好处的颤音，眼角那滴倔强着不肯落下的泪珠，
微微绞紧的手指关节泛着一丝发白——
每一个细节都精准得像是……排练过的。
虽然他自己学不会, 但他在职场见过太多这样的人，但眼前这位，段位显然更高。
```
→ 用角色视角的观察代替作者的评价

## 速查表

| 反模式 | 检查方法 |
|--------|----------|
| Tell > Show | 是否直接说"他是个XX的人"而非展示？ |
| 心理扁平 | 角色有没有情感反应，哪怕是被压制的？ |
| 感官单一 | 这个场景有几种感官描写？至少要2种 |
| 对话功能化 | 对话有没有动作/神态/停顿？ |
| 时间跳跃 | 跳跃前有没有铺垫或过渡？ |
| 战斗技术化 | 有没有紧张感和角色情绪？ |
| 评价过多 | 有没有"太XX了"、"简直"等词？ |
